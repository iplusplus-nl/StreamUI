The Math

How four points define a smooth transition.

The Cubic Expression

A cubic Bézier curve uses parametric equations based on a time parameter \( t \) that goes from 0 to 1. The position \( B(t) \) is interpolated between four points:

\[ B(t) = (1-t)^3 P_0 + 3(1-t)^2 t P_1 + 3(1-t) t^2 P_2 + t^3 P_3 \]
  • \( P_0 \) (Start): Always fixed at (0, 0) in CSS easings.
  • \( P_1 \) (Control 1): Pulls the curve initially. Determines starting acceleration.
  • \( P_2 \) (Control 2): Pulls the curve near the end. Determines deceleration.
  • \( P_3 \) (End): Always fixed at (1, 1) in standard easings.
  • \( t \) (Time): Progress through the animation.
